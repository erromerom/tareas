Primera curva de titulación

Calcule la curva de distribución de especies de los complejos del ion plata con amoniaco en función del pH. Asuma que \(\require{mhchem}c(\ce{Ag+}) =\) 0.0100 mol/L y \(c(\ce{NH3}) =\) 0.100 mol/L.

Reacciones

\[ \begin{array}{rcll} \ce{Ag^{+} + NH3} & \ce{<=>} & \ce{Ag(NH3)^+} & \log \beta_1 = 3.32\\ \ce{Ag^+ + 2 NH3} & \ce{<=>} & \ce{Ag(NH3)2^+} & \log \beta_2 = 7.24\\ \end{array} \]

beta.1 <- 10^3.32
beta.2 <- 10^7.24

Dependencia con el pH del amoniaco

\[\require{mhchem} \begin{array}{rcll} \ce{NH4^+ + H2O} & \ce{<=>} & \ce{MH3 + H3O+} & \text{p}K_{\text{a}} = 9.24\\ \end{array} \]

alp <- function(pH, pKa){
  n <- length(pKa)
  num <- 10^-(pH * (n:0) + cumsum(c(0,pKa)))
  num / sum(num)
}
pKa <- c(9.24)

Balance de masas

\[\require{mhchem} \begin{array}{rll} c(\ce{Cu^2+}) &=\displaystyle 0.0100 \times \cfrac{10.00}{20.00 + V} &= [\ce{Cu^2+}] + [\ce{CuY^2-}]\\ c(\ce{Zn^2+}) &=\displaystyle 0.0100 \times \cfrac{10.00}{20.00 + V} &= [\ce{Zn^2+}] + [\ce{ZnY^2-}]\\ c(\ce{EDTA}) &=\displaystyle 0.0100 \times \cfrac{V}{20.00 + V} &= \cfrac{[\ce{Y^4-}]}{\alpha(\ce{Y^4-})} + [\ce{CuY^2-}] + [\ce{ZnY^2-}]\\ \end{array} \]

\[\require{mhchem} \begin{array}{rl} \alpha(\ce{Cu^2+}) &=\displaystyle \cfrac{1}{1 + K_{\text{f}}(\ce{CuY^2-})\times [\ce{Y^4-}]} \\ \alpha(\ce{Zn^2+}) &=\displaystyle \cfrac{1}{1 + K_{\text{f}}(\ce{ZnY^2-})\times [\ce{Y^4-}]} \\ \end{array} \]

curva <- function(pY, V){
  Y <- 10^-pY
  aCu <- 1 / (1 + beta.CuY * Y)
  aZn <- 1 / (1 + beta.ZnY * Y)
  Cu  <- aCu * 0.1 / (20 + V)
  Zn  <- aZn * 0.1 / (20 + V)
  CuY <- beta.CuY * Cu * Y
  ZnY <- beta.ZnY * Zn * Y
  EDTA <- 0.01 * V / (20 + V)
  EDTA - Y / aY - CuY - ZnY
}

Definiendo la curva de titulación

El volumen se calculará entre 0.05 y 25.00 mL con intervalos de 0.01 mL.

V <- seq(0.05, 25.00, by = 0.01)

Calculando pY y pM

Se calcula el valor de pY para todos los valores de titulante

pY <- sapply(V, function(x)
  uniroot(curva, V = x, interval = c(0, 200), tol = 1e-14)$root)

Con el valor de pY se calcula las concentraciones en equilibrio de los cationes de cobre y cinc.

Y <- 10^-pY
aCu <- 1 / (1 + beta.CuY * Y)
aZn <- 1 / (1 + beta.ZnY * Y)
pCu <- -log10(aCu * 0.1 / (20 + V))
pZn <- -log10(aZn * 0.1 / (20 + V))
